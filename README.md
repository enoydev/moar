## Instalação

1) Clone o repositório diretório desejado
   ```
      git clone https://github.com/enoydev/moar.git
   ```

2)Crie sua Env
```
python3 -m venv env
```

3) Inicie a env
```
source env/bin/activate
```

  
4) Instale as dependências no diretório com a env iniciada:

```
 pip install -r requirements.txt
```

## Execute app.py

```
python app.py
```
